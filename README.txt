Описание проекта
веб-приложение, позволяющее пользователям загружать, хранить и управлять файлами через интернет-браузер.
Проект предоставляет базовый функционал для работы с файлами, включая их загрузку, скачивание, удаление и организацию в папках.
Цель проекта
-Загрузка, хранение и скачивание файлов
-Управление файлами через веб-интерфейс
-Аутентификация пользователей
